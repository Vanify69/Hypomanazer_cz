Nastavení rozšíření

Pro vývoj nech prázdné nebo nastav http://localhost:4000. V produkci zadej adresu API (např. z Railway). Bez správné URL nebude spárování fungovat.

Po zapnutí se v popupu rozšíření zobrazí sekce „Procházet DOM“ – procházení stránky banky a export mapování polí do mapping packu.

Heslo se neukládá v čitelné podobě. V popupu se admin sekce zobrazí až po zadání hesla.
